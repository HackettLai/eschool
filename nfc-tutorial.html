設定教學

請點選以下按鈕或自行輸入以下連結

按此打開網頁

*本技術由 AMT Limited 提供，歡迎您使用。

1 - 填寫資料

然後可於網頁下方的表格填寫個人資料

1留意聯絡人 姓名資訊 是 必填 欄位

2可按需要增加項目

2 - 檢查資料

確認資料無誤後於表格下方：

1點擊 I'm not a robot 完成reCAPTCHA人機驗證

2然後再點擊 立即建立卡片 以開始制作您的個人智能名片專屬網址

3 - 取得網址

制作成功後，彈出式視窗中將會顯示您的個人智能名片專屬網址，請記下此網址，你亦可：

1點擊 複製連結 以複製網址

2最後點擊 完成與關閉 以關閉視窗

41 - 下載手機應用程式

請根據您使用的手機系統，下載免費NFC寫入程式 （NFCTool）。

52 - 寫入資料

打開程式 （NFCTool），根據以下示範影片把您的個人智能名片專屬網址 網頁地址寫入到NFC中。

  1. 點選 Write
  2. 點選 Add a record
  3. 點選 URL / URI
  4. 輸入您的 個人智能名片專屬網址 網頁地址
  5. 點選 OK
  6. 點選 Write / XX Bytes
  7. 把NFC靠近到手機接收器
  8. 完成！

63 - 如何使用

只需使用NFC與對方手機觸碰，對方便可以立即於手機打開網址，然後儲存您的聯絡資料！
