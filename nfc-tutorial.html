主要功能

個人電子名片

與對方手機觸碰後即可將您的個人電子名片顯示於對方手機上

感應打開網頁

將網址寫入標籤，只要與對方手機觸碰即可打開網頁內容

其他更多功能*

包括：記錄文本、文件、電郵記錄、WiFi配置、藍牙連接⋯⋯等

*基於iOS系統限制，iPhone用戶只能讀取NFC上的網頁連結資訊。

1Web URL 供日後修改資料

2VCF File 供我們制作NFC名片之用

41 - 下載手機應用程式

請根據您使用的手機系統，下載免費NFC寫入程式 （NFCTool）。

52 - 寫入資料

打開程式 （NFCTool），根據以下示範影片把您的VCF File 網頁地址寫入到NFC中。

  1. 點選 Write
  2. 點選 Add a record
  3. 點選 URL / URI
  4. 輸入您的 VCF File 網頁地址
  5. 點選 OK
  6. 點選 Write / XX Bytes
  7. 把NFC靠近到手機接收器
  8. 完成！

63 - 如何使用

只需要使用NFC與對方手機觸碰，對方只需於手機打開網址，便可立即儲存您的聯絡資料！
