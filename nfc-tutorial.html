eSchool x AMT NFC 智能標籤使用教學

感謝您在「學與教博覽2024」與我們接洽，您的支持對我們意義重大。為表達我們的誠摯謝意，特別贈送您一份 eSchool x AMT NFC 智能標籤。隨附的設置教學旨在協助您輕鬆運用此科技，提升工作效率。

主要功能

個人電子名片

與對方手機觸碰後即可將您的個人電子名片顯示於對方手機上

感應打開網頁

將網址寫入標籤，只要與對方手機觸碰即可打開網頁內容

其他更多功能*

包括：記錄文本、文件、電郵記錄、WiFi配置、藍牙連接⋯⋯等

*基於iOS系統限制，iPhone用戶只能讀取NFC上的網頁連結資訊。

設定教學

請點選以下按鈕或自行輸入以下連結(https://vcard.link/card)以開啟網頁。

1 - 建立檔案

1輸入檔案名稱

2及3請記下填寫之資料供日後修改之用

2 - 填寫資料

然後可於下方表格填寫個人資料

1可按需要增加項目

2完成後按 Create Card以提交表格

3 - 檢查資料

確認資料無誤後於頁面右上方點擊Share

於彈出式視窗中記下上圖中標示的網址，其中：

1Web URL 供日後修改資料

2VCF File 供我們制作NFC名片之用

4 - 下載手機應用程式

請根據您使用的手機系統，下載免費NFC寫入程式 （NFCTool）。

5 - 寫入資料

打開程式 （NFCTool），根據以下示範影片把您的VCF File 地址寫入到NFC中。

  1. 點選Write
  2. 點選Add a record
  3. 點選URL / URI
  4. 輸入您的VCF File 地址
  5. 點選OK
  6. 點選Write / XX Bytes
  7. 把NFC靠近到手機接收器
  8. 完成！

6 - 如何使用

只需要使用NFC與對方手機觸碰，對方只需於手機打開網址，便可立即儲存您的聯絡資料！
